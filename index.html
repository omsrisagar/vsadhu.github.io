Research Projects (PhD)
Decision Making Under Uncertainty (PhD Thesis)
Title: Real-time Autonomic Decision Making Under Uncertain Environments [Robots/Drones/UAVs]
Collaborators: Prof. Roberto Tron, Prof. Saman Zonouz, Dr. Teruhisa Misu
+ Developing novel Multi-Agent Deep Reinforcement Learning (MADRL) framework (distributed
multi-agent actor-critic approach) for drones for autonomous aerial exploration e.g., victim search in
disaster scenarios.
+ Proposed a reactive framework for unpredictable uncertainties via Convolutional-LSTM deep learning
network classifier [Tensorflow] to identify anomalous events (e.g., propeller breakdown) from sensor
data for later planning.
+ Tested classifier by creating crash-like scenarios in Microsoft’s AirSim Drone Simulator [C++] as
well as on real drones.
Title: Decentralized Multi-Agent Reinforcement Learning (MARL) for Disaster Scene Data Collection
Collaborators: Prof. Saman Zonouz
+ Designed MARL framework [Python/Android] to direct autonomously bystanders/robots in a
coordinated manner for data collection to build 3D maps of the disaster scene.
+ Working on preserving privacy and developing decentralized Partially Observable Markov Decision
Process (Dec-POMDP) for an online solution.
Smartphone Context Acquisition and Validation
Title: Smartphone Future Context/Location Prediction via HMM Based Collaborative Filtering
Collaborators: Prof. Saman Zonouz
+ Developed a novel emission model for Collaborative-Filtering Hidden Markov Model [Python] to
predict the phone’s current/future context (e.g., location) based on owner’s past personal/group
behavior (e.g., sequential patterns). Worked on real-life dataset.
+ Useful for mobile assistant technologies (Apple Siri, Google Now, etc.) to plan ahead or to validate
raw sensor data.
+ Detecting phone theft based on NLP Viterbi Decoding. Privacy preserved via Multi-Party Homomor-
phic Encryption.
Title: Privacy-preserving Multi-Modal Smartphone Localization via Collaborative Information Fusion
Collaborators: Prof. Saman Zonouz
+ Proposed novel privacy-aware cross-building (pervasive) room-level localization solution ( 1m accu-
racy) [custom Android app].
+ Designed own classifier [R], performing better than Multinomial Logistic Regression.
Low-power and Low-cost Wireless Sensing
Title: Energy-efficient Analog Sensing for Large-scale and High-density Persistent Wireless Monitoring
+ Proposed a dumb-sensing and smart-processing architecture for WSNs that splits sensing and
computational tasks between energy-efficient low cost substrate sensors and digital nodes.
+ Proposed and designed via circuit a Field Effect Transistor (FET) based sensor encoding technique
to realize low-power and low-complexity substrate sensors. Proposed a corresponding decoding
2/11
technique at the receiver.
+ Proposed a Kernel Density Estimation (KDE) and Kullback-Leibler Divergence (KLD) based
technique to estimate source distribution at the receiver to optimize sensor encoding parameters.
